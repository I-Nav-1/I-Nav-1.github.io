About Me

I have hands-on experience in Linux system hardening, including securing user and group permissions, enforcing least-privilege access, and hardening SSH by disabling root login. I implement system-level security controls such as service minimization, secure file permissions, and audit logging. I also configure Data Loss Prevention (DLP) settings to enforce security controls, and applying policies to minimize data exfiltration and unauthorized user activity. Additionally, I apply kernel and operating system hardening practices, such as sysctl tuning, password policies, and log monitoring, to reduce the attack surface and ensure compliance with security best practices. I continuously monitor system logs and system behavior to identify anomalies and apply corrective hardening measures. I have experience setting up Zabbix monitoring, automating tasks using SaaS tools like Freshservice, managing ESXi virtualization environments, and handling user access management across various SaaS platforms.
